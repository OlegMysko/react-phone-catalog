# Product Catalog App

A modern **Product Catalog** application built with **React, TypeScript, and Redux**.
This project demonstrates global state management, routing, async data fetching, and scalable frontend architecture.

## üîß Technologies
- React
- TypeScript
- Redux / Redux Toolkit
- React Router
- SCSS Modules
- LocalStorage

##  Key Features
- Product catalog (Phones / Tablets / Accessories)
- Sorting and pagination synced with URL
- Product details page
- Shopping cart with total calculation
- Favorites functionality
- State persistence via localStorage
- Loaders, error handling, empty states
- NotFoundPage for unknown routes

## Architecture
- Feature-based structure (`modules`)
- Isolated components with CSS Modules
- Global state handled by Redux
- Clear separation of UI, logic, and pages

## Redux is used for
- Cart state
- Favorites state
- Products data with loading and error states
